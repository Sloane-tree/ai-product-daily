## 💡 Today's Product Insight

**Skills 正在成为 Agent 时代的 App Store。**

本周 GitHub Trending 前十中，Skill 仓库占据了显著位置。caveman（82K ⭐，让 AI "变懒"来省 65% token）、superpowers（245K ⭐，Agent 技能框架）、claude-skills（21K ⭐，337+ 模块化技能）——这些不是模型，不是工具，而是"如何用好 Agent"的可复用知识包。

这意味着：Agent 生态的价值链正在从"模型层"上移到"技能层"。就像 iOS App Store 让开发者不需要懂芯片就能创造价值，Skills 让 PM 和领域专家不需要懂 Prompt Engineering 也能构建 Agent 能力。

对产品经理的核心启示：**不要再花时间写 Prompt 了，开始积累 Skills。** 每个领域的专业知识（如何做竞品分析、如何设计用户访谈、如何写 PRD）都可以封装为可复用的 Skill，然后被 Agent 按需调用。这不是效率工具，这是知识资产的数字化。
